# ADR-0003: Konfiguracja projektu w YAML

- Status: Accepted
- Data: 2026-07-21

## Decyzja

Każdy projekt SLR ma własny plik `config.yaml`, przechowywany w Git.

## Uzasadnienie

YAML jest czytelny dla badacza, łatwy do wersjonowania i nadaje się do zapisu:
- pytań badawczych;
- strategii wyszukiwania;
- źródeł;
- zakresu lat;
- kryteriów włączenia i wyłączenia.

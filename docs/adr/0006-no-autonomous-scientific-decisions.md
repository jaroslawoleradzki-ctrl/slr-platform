# ADR-0006: Brak autonomicznych decyzji naukowych

- Status: Accepted
- Data: 2026-07-21

## Decyzja

AI nie może samodzielnie:
- ostatecznie wykluczać publikacji;
- zatwierdzać oceny jakości;
- rozstrzygać niejednoznacznych duplikatów;
- zmieniać protokołu SLR.

AI może generować rekomendacje wraz z uzasadnieniem, poziomem pewności i odniesieniem do danych źródłowych.
